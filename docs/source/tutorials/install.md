## Install

### Install from PyPI

```shell

```

### Install from GitHub

```shell
pip install 'git+https://github.com/YuxinZhaozyx/easycore.git'
```

### Install from a local clone

```shell
git clone https://github.com/YuxinZhaozyx/easycore.git
cd easycore
pip install -e .
```
